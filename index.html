FINDMe

FindME

FINDME is a python script used in Recon to find missing people online from SAPS website. FINDME scrapes user information of Missing person using User ID from this URL https://www.saps.gov.za/crimestop/missing/detail.php?bid=. I created this tool to help Forensic Investigators, people from Tracelabs or the charleyproject to trace people efficiently and get the output of Information in a nice HTML Format and enhancing OSINT. Socuma

Socuma is a Python OSINT (Open-Source Intelligence) tool used for finding online profiles associated with a given username on various social media and online platforms. The tool is still undergoing maintenance to fix false positives and trying to fix capture challenges on some websites.

Cobvis

Cobvis

Cobvis is a Python script used to find and extract email addresses from a target website. This script takes a target URL, recursively fetches web pages linked from that URL, extracts email addresses from the content of these pages, and prints the extracted email addresses to the console.
